Enum jumpy::prelude::Visibility

pub enum Visibility {
    Inherited,
    Hidden,
    Visible,
}
Expand description

User indication of whether an entity is visible. Propagates down the entity hierarchy.

If an entity is hidden in this way, all Children (and all of their children and so on) who are set to Inherited will also be hidden.

This is done by the visibility_propagate_system which uses the entity hierarchy and Visibility to set the values of each entity’s ComputedVisibility component.

Variants§

§

Inherited

An entity with Visibility::Inherited will inherit the Visibility of its Parent.

A root-level entity that is set to Inherited will be visible.

§

Hidden

An entity with Visibility::Hidden will be unconditionally hidden.

§

Visible

An entity with Visibility::Visible will be unconditionally visible.

Note that an entity with Visibility::Visible will be visible regardless of whether the Parent entity is hidden.
